⚙️ Configuration sudoers recommandée

        
Pour que l'application panel puisse exécuter les commandes système sans mot de passe, ajoutez ces règles via visudo :

        
www-data ALL=(ALL) NOPASSWD: /bin/systemctl start *
www-data ALL=(ALL) NOPASSWD: /bin/systemctl stop *
www-data ALL=(ALL) NOPASSWD: /bin/systemctl restart *
www-data ALL=(ALL) NOPASSWD: /bin/systemctl enable *
www-data ALL=(ALL) NOPASSWD: /bin/systemctl disable *
www-data ALL=(ALL) NOPASSWD: /bin/systemctl daemon-reload
www-data ALL=(ALL) NOPASSWD: /bin/systemctl reload nginx
www-data ALL=(ALL) NOPASSWD: /usr/sbin/nginx -t
www-data ALL=(ALL) NOPASSWD: /bin/mkdir -p /etc/nginx/*
www-data ALL=(ALL) NOPASSWD: /bin/mkdir -p /opt/*

        
Remplacez www-data par l'utilisateur qui exécute gunicorn si différent.
